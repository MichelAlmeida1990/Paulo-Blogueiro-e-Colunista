# Paulo Blogueiro e Colunista

Projeto de site para Paulo Agostinho, blogueiro e colunista.

## Descrição

Este projeto contém o site desenvolvido para o blogueiro e colunista Paulo Agostinho.

## Estrutura do Projeto

- `index.html` - Página principal
- `admin.html` - Painel administrativo
- `admin-dashboard.html` - Dashboard do administrador
- `public/` - Arquivos públicos (imagens, CSS, JS)
- `src/` - Código fonte

## Instalação

1. Clone o repositório
2. Instale as dependências com `npm install`
3. Inicie o servidor de desenvolvimento

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Node.js
